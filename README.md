ios初期化方法
このアプリを動かすには下記のコマンドをターミナルで実行しPCにCarthageを導入する必要があります。

$ ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

$ brew update

$ brew install carthage

$ carthage update --platform iOS
